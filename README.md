# AWS_Agent_Fabric

Production-ready AI Agent platform built on AWS.

## Project Goal

The project evolves from a simple AI Chat Agent into a complete AI Agent Fabric, including:

- AI Chat Agent
- Document Agent
- Knowledge Agent (RAG)
- Tool Agent
- Business Agent
- Multi-Agent System
- Self-Developing AI Agent Fabric

The project is developed incrementally using Git Flow and Semantic Versioning.
